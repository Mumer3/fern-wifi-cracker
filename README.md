# fern-wifi-cracker
Fern WiFi Cracker is a wireless security auditing tool used to test the strength of Wi-Fi networks

It helps identify vulnerabilities in Wi-Fi security (like WEP, WPA, WPA2) by attempting to crack passwords using known techniques. Security professionals use it to check if a network can be easily breached.
